How to Use Christmas Label Maker

Quick Guide:

  1. Select a Contact: Choose a name from the dropdown menu.
  2. Manage Christmas List: Use the Add/Remove button to update your Christmas list.
  3. Create Labels: Labels automatically appear for Christmas list members.
  4. Print or Download:
    • Click 'Print Labels' to print all visible labels
    • Use the 'Download' button above each label to save individual labels
