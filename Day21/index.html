How to Use This Dashboard

View Modes

  • Global Rankings: See all toys ranked by total requests worldwide
  • Regional Analysis: Break down toy requests by geographic location
  • By Category: Group toys by type (using emoji categories)

Sorting Options

  • Sort by Quantity: Order toys by number of requests (highest to lowest)
  • Sort Alphabetically: Order toys by name (A to Z)
  • Sort by Category: Group toys by their emoji category

Dashboard Features

  • Most Requested: Shows the toy with the highest total requests
  • Total Requests: Displays the sum of all toy requests globally
  • Interactive Cards: Hover over toy cards to highlight them
